# 四种编码的转换

二级计算机的原码、反码、补码和偏移码的转换

## 目录

- [简介](#简介)
- [功能特性](#功能特性)
- [安装](#安装)

## 简介

一个练习使用的小脚本。
用于在全国二级计算机考试中常见的四种二进制编码的相互转换。

## 功能特性

- 没什么特殊的，就是简单的二进制编码转换
- GUI可视化用户交互

## 安装

把.jar文件下载下来，在jdk21环境下运行就好。懒得打包成.exe文件。哈哈。

### 系统要求

- 三端互通：Windows、Mac OS、Linux
- jdk21

### 安装步骤

下载下来就能直接用。
